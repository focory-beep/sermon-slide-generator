# 🚀 시작하기 - 클라우드 배포

## 빠른 시작 (5분)

### 1. GitHub에 업로드

**가장 쉬운 방법 (드래그 앤 드롭):**

1. [github.com](https://github.com) 로그인
2. 우측 상단 `+` → `New repository`
3. Repository name: `sermon-slide-generator`
4. Public 선택 → `Create repository`
5. `uploading an existing file` 클릭
6. **이 폴더의 모든 파일을 드래그 앤 드롭**
7. `Commit changes` 클릭

✅ GitHub 업로드 완료!

---

### 2. Render에 배포

1. [render.com](https://render.com) 로그인 (GitHub 계정 사용 가능)
2. `New +` → `Web Service`
3. GitHub 저장소 연결 → `sermon-slide-generator` 선택
4. 설정:
   - Name: `hamburg-church-slides` (원하는 이름)
   - Region: `Frankfurt (EU Central)`
   - Plan: `Free`
5. `Create Web Service` 클릭
6. 3-5분 기다리기...

✅ 배포 완료!

---

### 3. 접속하기

상단의 URL 클릭:
```
https://hamburg-church-slides.onrender.com
```

⏳ 첫 접속 시 30초 기다리기 (무료 티어 한계)

그 다음부터는 빠름!

---

## ✨ 테스트

1. **성경 참조**: `요3:16` 입력 (약어!)
2. **번역본**: `개역개정` 선택
3. **본문**: 성경 구절 복사-붙여넣기
4. **PPT 생성하기** 클릭

→ PPT 다운로드 확인! 🎉

## 💡 사용 팁

### 북마크 추가
브라우저에 북마크 저장:
```
예배 슬라이드 생성기
https://YOUR-APP-NAME.onrender.com
```

### 팀원과 공유
```
예배 PPT 자동 생성 도구:
https://hamburg-church-slides.onrender.com

약어 사용 가능:
- 요3:16 → 요한복음 3:16
- 창1:1 → 창세기 1:1
- 롬8:28-30 → 로마서 8:28-30
```
